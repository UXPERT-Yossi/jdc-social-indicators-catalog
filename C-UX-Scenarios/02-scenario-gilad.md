# גלעד המדידן — בוחן את הארכיטקטורה הסמנטית של הקטלוג

**ארכיטיפ:** גלעד המדידן — עדיפות שנייה
**כניסה:** מגיע כמשתמש ראשון — כנראה שותף בתהליך בניית הקטלוג. פותח מהלפטופ. מצב רגשי: ביקורתי ומדוקדק — "אם הארכיטקטורה לא נכונה, אדע מיד."
**מטרה:** לאמת שהקטלוג מייצג נכון את עולם המדידה — שהמבנה, הדיוק והקשרים הסמנטיים ראויים לחתימתו

---

## מסכים

1. **דף הבית**
   גלעד נוחת ומחליט מיד — הוא לא מחפש מדד ספציפי, הוא רוצה לראות את המבנה. לוחץ על **עיון** (לא על שדה החיפוש). הוא חושב במערכות, לא בשאילתות.

2. **עץ ניווט — ממד**
   רואה את ארבעת הממדים: תשומות / תפוקות / תוצאות / אימפקט. פותח "תוצאות". בוחן את ההגיון ההיררכי — האם חלוקת תת-התחומים הגיונית? האם הסדר נכון?

3. **עץ ניווט — תחום**
   נכנס לתחום "רווחה רגשית". רואה רשימת מדדים עם ספירה. בוחן את הכיסוי — האם כל המדדים הרלוונטיים כאן? מה חסר?

4. **כרטיס מדד**
   פותח מדד שהוא מכיר לעומק. קורא את ההגדרה — בודק דיוק מילה במילה. בוחן את המתודולוגיה, המקורות, ואת הקשרים הסמנטיים. כרטיס מדויק.

5. **Node Chart — מפה סמנטית**
   עובר למצב הגילוי הויזואלי. הגרף מציג את הקשרים שבנה — הם שם, ומיושמים נכון. מרחיב צומת לתחום סמוך. מגלה חיבור בין שני מדדים שלא ראה עד כה — הגרף לימד אותו משהו חדש.

6. **כרטיס מדד (מתוך Node Chart)** ✓ הצלחה
   פותח מדד שהגיע אליו דרך הגרף. מאמת את ההגדרה והקשרים. הכרטיס מדויק. גלעד מרוצה — הקטלוג מייצג נכון את עולם המדידה כפי שהוא מאמין בו.
